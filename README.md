## ?? Descripción

Sistema de gestión de pedidos y ventas para pequeńas y medianas empresas.

### ? Características

- ?? Gestión completa de órdenes (CRUD)
- ?? Facturas profesionales en PDF
- ?? Reportes de ventas (día/semana/mes)
- ?? Top productos y clientes frecuentes
- ?? Búsqueda y filtros en tiempo real
- ?? Backups automáticos
- ?? Envío de facturas por email
- ?? Atajos de teclado
